Team 06 Open-source Project

Project Objective:
The team wants to resolve the issue of adding red/green colorblind-friendly colors and the issue of allowing more game-speed control.
The team also wants to add the instant block-drop feature as inspired by the original Tetris game.

List of Implementations:

	files: js/input.js, js/main.js

Implementation 2: toggleSpeed function
	files: js/input.js

Implementation 3: toggleDrop function
	files: js/input.js
